anchovy.js

What is anchovy.js?

anchovy.js is a small JavaScript library that allows you to create dynamic web pages. It is inspired by the Vue.js framework and the Batman.js library.

Some examples

How to use it?

You can download the library from here and include it in your HTML page: 

<script src="anchovy.js"></script>
